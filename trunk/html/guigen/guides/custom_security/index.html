Guida alla realizzazione della Custom Security

Introduzione

In guigen è possibile realizzare l'integrazione con un ampio spettro di meccanismi di security. Questo obiettivo si ottiene mediante la modellazione di:

  • un modello che descrive il meccanismo di security, denominato SecurityProfile
  • un elemento CustomSecurity nel SecurityModel dell'applicazione

Risorse

Per l'utilizzo del meccanismo della CustomSecurity sono disponiibli le seguenti risorse:

  • Descrizione dell'architettura e manuale di utilizzo, in formato PDF
  • Esempio di applicazione (progetto generatore e generato)
